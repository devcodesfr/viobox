- [ ] Public GitHub repo is available.
- [ ] README explains setup and current project status.
- [ ] Bot calls only the approved assessment test number.
- [ ] At least 10 calls are completed before final submission.
- [ ] Call recordings are saved.
- [ ] Transcripts are saved.
- [ ] Bugs and quality issues are documented.
- [ ] AI usage is documented.
- [ ] Loom walkthrough video is recorded.
- [ ] AI-debugging / AI-development recording is captured.
- [ ] Final bug report avoids unverified or fake findings.
